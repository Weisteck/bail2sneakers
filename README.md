# Installation
```shell
yarn
```

# Development

## Requirements
- [MongoDB](https://www.mongodb.com/try/download/community)
- [Yarn](https://classic.yarnpkg.com/en/docs/install/#windows-stable)
- [Node.js & NPM](https://nodejs.org/en/download/)

### start front
````shell
yarn start:front
````

### start back
````shell
yarn start:back
````
